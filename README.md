# DND Client
为DND玩家设计的网页客户端。
## 概述 
该项目目前的目标是为DND5e提供一个现代化的、基于网页的车卡工具。附带目标则为对DND5e的规则进行结构化的存储。目前项目提供了一个测试用的网页，可以通过以下网址访问：[https://dnd5e.nicklin.work/create-character](https://dnd5e.nicklin.work/create-character)，如果在测试过程中发现了任何问题，欢迎与我联系。
